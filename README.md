# PodgyMonkey HA Add-ons

Custom Home Assistant add-ons.

## Installation

1. In Home Assistant: **Settings → Apps → Install app → ⋮ → Repositories**
2. Add: `https://github.com/PodgyMonkey/ha-addons`
3. The apps below will appear in the store

## Add-ons

### DokuWiki

A simple, fast wiki that stores all data in plain text files — no database required.

- Data persists in `/share/dokuwiki` and survives add-on reinstalls
- Accessible via the HA sidebar through ingress (no port exposure needed)
- Configurable PHP memory limits, upload limits, and timezone
